Rename Folder in Git Bash

To rename a folder in Git Bash git mv command is used. The git mv is followed by the old name of the folder and the new name of the folder.
The steps & the commands are as follows:

Steps:

  • Open the Git Bash.
  • By using cd command change the current working directory to the local Git repository
  • Type git mv followed by the old name of the folder and the new name of the target folder.
  • Type git status to confirm if the folder name has been successfully changed
  • Commit the changes made to the folder name.
  • Push the changes using "git push origin branch_name"

The commands below:


    git mv old_folder_name new_folder_name
    git status
    git commit -m "Renamed old_folder_name to new_folder_name"
    git push origin branch_name
